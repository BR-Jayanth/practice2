HTML doctype

In HTML, the DOCTYPE (Document Type Declaration) serves two crucial purposes:
1. Informs the browser about the HTML version:
The DOCTYPE tells the browser which version of HTML the document is
written in. This helps the browser interpret the code correctly and
render the page according to the specific version's rules.
2. Enables standards mode:
By specifying the HTML version, the DOCTYPE activates "standards
mode" in the browser. This mode ensures that the browser renders the
page according to the official HTML and CSS specifications, leading
to more predictable and consistent behavior across different browsers.
