HTML body

The HTML <body> tag defines the main content of a web page. It is placed inside the <html> tag, after the <head> tag. All of the visible content of a web page, such as text, images, hyperlinks, tables, lists, and forms, is placed inside the <body> tag.

The <body> tag has several attributes that can be used to control the appearance of the web page, such as the background color, the text color, and the font family.
